Bu soru özellikle matematikçiler için:

X herhangi bir tamsayı olmak üzere, X tane X in toplamı, 

    X + X + X + ... + X = X * X,
               X tane X = X2,
    X + X + X + ... + X = X2,
Her iki tarafın türevini alırsanız; 
d                         d
--(X + X + X + ... + X) = -- X2,
dx                        dx

    1 + 1 + 1 + ... + 1 = 2*X,
               X tane 1 = 2*X,
                      X = 2*X,
X sıfır olmadığı için her iki taraf X e bölünebilir ve böylece de, 
                      1 = 2 elde edersiniz.
