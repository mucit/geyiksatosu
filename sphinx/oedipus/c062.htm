Kitabın fiyatı X, benim cebimdeki para A ve arkadaşımın cebindeki para B olsun. Şu ifadeleri yazalım:

X - 20,000 = A ve
X - 240,000 = B

Paraları birleştirdiğimizde toplam paramız,

A + B = X - 20,000 + X - 240,000 = 2X - 260,000 olur.

Bu para kitabın fiyatından az olduğuna göre,

2X - 260,000 < X,
X - 260,000 < 0 ve
X < 260,000 olur.

Demek ki kitabın fiyatı 260,000 TL den azdır. Bu durumda şu seçenekler bulunuyor:

Kitap Arkadasta Bende
255,000 235,000 15,000
250,000 230,000 10,000
245,000 225,000 5,000
240,000 220,000 0

Başka bilgi verilmediğine göre bunlardan biridir.
